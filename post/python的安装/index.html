Python的安装

学Python第一件事当然是要安装Python啦

Python安装应该很简单，在Python官网下载就行了，我下载的是3.8.6的版本，因为在Tensorflow上看到Tensorflow支持的最新版是3.8。为了以后搞机器学习，先下着3.8.6的版本试试看。反正到后面也能升级。 下载下来之后一路点安装就行，这个没多大的难度。 其实我用Mac电脑可以用brew下载的，我之前也用brew下过python，但是后面再装其他包的时候老是遇到找不到位置的问题，后面实在没办法，把原来用brew下载的删掉了，换用这种常规的下载方式安装。 安装好之后在终端里面敲python就会显示版本，我敲了之后发现还是系统的2.7的版本。百度了才知道还要配置一下环境变量 Mac的环境变量配置就是修改个人目录下的.bash_profile文件，把下面这些加上去就行。

# Setting PATH for Python 3.8
# The original version is saved in .bash_profile.pysave
export PATH="/Library/Frameworks/Python.framework/Versions/3.8/bin:${PATH}"
export PATH="/usr/local/opt/python@3.8/bin:$PATH"
alias python="/Library/Frameworks/Python.framework/Versions/3.8/bin/python3.8"
alias pip="/Library/Frameworks/Python.framework/Versions/3.8/bin/pip3.8"

然后保存 然后在终端运行一下source ~/.bash_profile 我嫌在终端敲open什么的命令麻烦，一般都是直接用记事本打开.bash_profile然后保存就行 接着再在终端敲python就会显示正确的python版本了 好的，简单的Python安装就玩成了，撒花！！！
